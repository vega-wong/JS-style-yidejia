# eslint-config-yidejia

## usage

```ruby
$ npm install eslint-config-yidejia
```

add `.eslintrc` file in your project

copy this code to your `.eslintrc` file

```json

  {
    "root": true,
    "extends": "yidejia",
    "env": {
      "browser": true
    },
    "rules": {
    }
  }

```

and then enjoy yourself
